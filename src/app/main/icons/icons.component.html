Youpez uses Carbon Design System Icons
We don't created representation page for the icons because this theme does not use all of them. Basically these icons are on-demand, so you can include just the selected icons that you want to use (smaller package size).
If you want use another icon library, you have to just include/install them, it is super easy.
Carbon icons are provided through a set of packages allowing the use of icons in multiple frameworks. Icons are supported in vanilla, React, Angular, and Vue.
